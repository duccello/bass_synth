Here is the Code, electronic layout and 3D design for a Monophonic, foot-controlled synth, designed to play bass while playing piano.

The code it´s quite simple given the use of the Teensy Audio Library, which already provides 
digital oscilators, effects, envelopes and all functions one might need to easily make a working digital instrument.
The options I found on the market were either not portable for gigging, or too expensive for what they were.

The synth has a simple UI, with 4 controls:
PEAK (resonance)
RELEASE
CUTOFF
BLEND (a mix of the SUB and SAW oscilators)

The HOLD button makes the notes to be sustained, otherwise the Release will control it´s duration.
